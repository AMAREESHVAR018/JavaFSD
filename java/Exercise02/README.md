# Exercise 2 - Simple Calculator

## Objective
Perform basic arithmetic operations.

## File Structure
- `SimpleCalculator.java`

## Sample Input
```text
Enter first number: 10
Enter second number: 5
```

## Sample Output
```text
Addition: 15.0
Subtraction: 5.0
Multiplication: 50.0
Division: 2.0
```

## Explanation
The program reads two numbers and prints the result of four basic operations.
