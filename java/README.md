# Core Java Exercises

Theme: Community Learning Management System

## Notes
- Exercises 1-28 run on Java 8+
- Exercise 29 runs on Java 16+
- Exercise 30 and Exercise 40 use Java 21 features
- JDBC exercises use SQLite with Maven
- Module, networking, and utility exercises include complete setup instructions in their own README files

## Folders
- `Exercise01` to `Exercise41`

## Quick Run
For a simple class:
```bash
javac HelloWorld.java
java HelloWorld
```

For Maven-based JDBC exercises, follow the README inside each folder.
