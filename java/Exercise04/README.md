# Exercise 4 - Leap Year Checker

## Objective
Check if a given year is a leap year.

## File Structure
- `LeapYearChecker.java`

## Sample Input
```text
Enter year: 2024
```

## Sample Output
```text
2024 is a Leap Year
```

## Explanation
A leap year is divisible by 400, or divisible by 4 but not by 100.
