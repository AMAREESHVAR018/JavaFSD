# Exercise 35 - TCP Client-Server Chat

## Objective
Build a simple TCP server and client.

## File Structure
- `Server.java`
- `Client.java`
- `README.md`

## Sample Output
Server:
```text
Server started on port 5000
Client connected
Received: Hello Server
```

Client:
```text
Connected to server
Server: Hello Client
```

## Execution Instructions
1. Start `Server.java`.
2. Run `Client.java`.
3. Type a message in the client console.

## Explanation
The server listens for one client connection and echoes a reply.
