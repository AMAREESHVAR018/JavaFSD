# Exercise 37 - Using javap to Inspect Bytecode

## Objective
Inspect a compiled class using `javap`.

## File Structure
- `BytecodeSample.java`

## Sample Output
```text
Bytecode inspection instructions
```

## Explanation
Compile the class, then use `javap -c` to view the bytecode.

## Commands
```bash
javac BytecodeSample.java
javap -c BytecodeSample
```
