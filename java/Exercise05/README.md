# Exercise 5 - Multiplication Table

## Objective
Print the multiplication table for a number.

## File Structure
- `MultiplicationTable.java`

## Sample Input
```text
Enter number: 5
```

## Sample Output
```text
5 x 1 = 5
5 x 2 = 10
...
5 x 10 = 50
```

## Explanation
The program uses a loop to print values from 1 to 10.
