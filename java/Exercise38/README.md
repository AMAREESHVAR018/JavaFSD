# Exercise 38 - Decompile a Class File

## Objective
Use JDK tools to inspect compiled class behavior.

## File Structure
- `DecompileSample.java`

## Sample Output
```text
Decompile with javap instructions
```

## Explanation
Compile the class and use `javap -c -p` to inspect the generated bytecode.

## Commands
```bash
javac DecompileSample.java
javap -c -p DecompileSample
```
